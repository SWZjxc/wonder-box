## 快速启动 🏁

要开始使用 Wonder-Box，您可以按照以下简单步骤进行操作:

1. 引入依赖坐标

   ```xml
   <dependency>
       <groupId>com.wonderbox</groupId>
       <artifactId>wonderbox-all</artifactId>
       <version>0.0.1</version>
   </dependency>
     ```
2. 使用它即可
